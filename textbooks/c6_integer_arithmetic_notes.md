# 一、邏輯移位 vs 算術移位（Logical vs Arithmetic Shifts）

這是整章最核心的觀念，務必先弄懂兩者「補位」的差別：

| 類型 | 補進來的新位元 | 用途 |
|------|--------------|------|
| **邏輯移位 Logical Shift** | 補 **0**（zero） | 處理**無號數 (unsigned)** |
| **算術移位 Arithmetic Shift** | 補 **符號位元的複製 (sign bit)** | 處理**有號數 (signed)** |

### 圖解

```
邏輯移位 (Logical Shift)：空出來的位置補 0
   0 ──► [ • • • • • • • • ] ──► CF

算術移位 (Arithmetic Shift)：空出來的位置補「符號位元」的複製
   ┌──┐
   │  ▼
   [ • • • • • • • • ] ──► CF
   （最高位元複製自己，符號被保留）
```

> 🔑 **核心差別**：邏輯移位永遠補 0；算術移位「向右」時會複製符號位元，所以負數移位後仍是負數。

---

# 二、SHL 指令（Shift Left，邏輯左移）

- **SHL**：對目的運算元做**邏輯左移**。
- 最低位元（LSB）補 **0**，最高位元（MSB）移出到 **CF（Carry Flag）**。

### 圖解

```
   CF ◄── [ • • • • • • • • ] ◄── 0
   (最高位移入 CF)        (最低位補 0)
```

### 語法
```asm
SHL  destination, count
```

---

## 應用：快速乘法（Fast Multiplication）

- **左移 1 位 = 乘以 2**
- **左移 n 位 = 乘以 2ⁿ**

### 例題 1：左移 1 位（×2）
```asm
mov dl, 5
shl dl, 1          ; DL = 10
```
```
Before:  0 0 0 0 0 1 0 1  = 5
After:   0 0 0 0 1 0 1 0  = 10
```

### 例題 2：左移 2 位（×2² = ×4）
```asm
mov dl, 5
shl dl, 2          ; DL = 20   （5 × 2² = 5 × 4 = 20）
```

---

# 三、SHR 指令（Shift Right，邏輯右移）

- **SHR**：對目的運算元做**邏輯右移**。
- 最高位元（MSB）補 **0**，最低位元（LSB）移出到 **CF**。

### 圖解

```
   0 ──► [ • • • • • • • • ] ──► CF
   (最高位補 0)        (最低位移入 CF)
```

### 應用：快速「無號」除法
- **右移 n 位 = 除以 2ⁿ**

### 例題
```asm
mov dl, 80
shr dl, 1          ; DL = 40   （80 ÷ 2）
shr dl, 2          ; DL = 10   （40 ÷ 2² = 40 ÷ 4）
```

### 例題（圖解）：32 ÷ 2 = 16
```asm
mov dl, 32
shr dl, 1
```
```
Before:  0 0 1 0 0 0 0 0  = 32
After:   0 0 0 1 0 0 0 0  = 16
```

> ⚠️ **注意**：SHR 是「邏輯」右移補 0，**只適用於無號數**。負數（有號數）要用 SAR！

---

# 四、SAL 與 SAR 指令（算術移位）

## SAL（Shift Arithmetic Left，算術左移）
- **SAL 與 SHL 完全相同**（identical）。
- 最低位元補 **0**。
- （因為左移時無號 / 有號的乘法結果一致，所以不需另外處理符號。）

```
   CF ◄── [ • • • • • • • • ] ◄── 0
```

## SAR（Shift Arithmetic Right，算術右移）
- **SAR**：對目的運算元做**算術右移**。
- 最高位元補回 **符號位元（sign bit）** → **保留數值的正負號**。

```
   ┌──┐
   │  ▼
   [ • • • • • • • • ] ──► CF
   (最高位複製符號位元)    (最低位移入 CF)
```

> 🔑 **重點句**：An arithmetic shift preserves the number's sign.（算術移位會保留數字的符號。）

### 例題：負數右移
```asm
mov dl, -80
sar dl, 1          ; DL = -40   （-80 ÷ 2）
sar dl, 2          ; DL = -10   （-40 ÷ 2²）
```

---

## 應用：快速「有號」除法（Fast Signed Division）

- 要把**有號數**除以 2 的次方，必須用 **SAR**（不能用 SHR）。

### 例題（圖解）：-128 ÷ 2³ = -16
```asm
mov dl, -128
sar dl, 3
```
```
Before:  1 0 0 0 0 0 0 0  = -128
After:   1 1 1 1 0 0 0 0  = -16
         ↑↑↑↑
      符號位元 1 被複製補回（保留負號）
```

> 💡 對比記憶：
> - **無號**除法 → 用 **SHR**（補 0）
> - **有號**除法 → 用 **SAR**（補符號位元）

---

## 🎯 例題演練（Your turn）

求每次移位後 AL 的十六進位值：

```asm
mov al, 6Bh
shr al, 1          ; a. al = ?
shl al, 3          ; b. al = ?
mov al, 8Ch
sar al, 1          ; c. al = ?
sar al, 3          ; d. al = ?
```

### 詳解

**a. `shr al,1`**（6Bh）
```
6Bh = 0110 1011
邏輯右移 1 位（補 0）：
      0011 0101 = 35h     ✅ a = 35h
```

**b. `shl al,3`**（接續 35h）
```
35h = 0011 0101
邏輯左移 3 位（補 0）：
      1010 1000 = A8h     ✅ b = A8h
```

**c. `sar al,1`**（8Ch，最高位=1 為負數）
```
8Ch = 1000 1100
算術右移 1 位（補符號位元 1）：
      1100 0110 = C6h     ✅ c = C6h
```

**d. `sar al,3`**（接續 C6h）
```
C6h = 1100 0110
算術右移 3 位（補符號位元 1，連補 3 個 1）：
      1111 1000 = F8h     ✅ d = F8h
```

| 小題 | 指令 | 答案 |
|------|------|------|
| a | shr al,1 | **35h** |
| b | shl al,3 | **A8h** |
| c | sar al,1 | **C6h** |
| d | sar al,3 | **F8h** |

# 五、ROL 指令（Rotate Left，向左旋轉）

- **ROL**：每個位元向左移。
- 最高位元（MSB）**同時**複製到 **CF** 與 **最低位元（LSB）**。
- **不會遺失任何位元（No bits are lost）**——移出去的位元繞回到另一端。

### 圖解

```
   ┌─────────────────────────┐
   │                         │
   CF ◄── [ • • • • • • • • ] ◄──┘
   (最高位 → CF，同時繞回最低位)
```

### 例題
```asm
mov al, 11110000b
rol al, 1          ; AL = 11100001b
                   ; 最高位 1 繞回到最低位

mov dl, 3Fh
rol dl, 4          ; DL = F3h
                   ; 3Fh=00111111 → 旋轉 4 位 → 11110011 = F3h
```

---

# 六、ROR 指令（Rotate Right，向右旋轉）

- **ROR**：每個位元向右移。
- 最低位元（LSB）**同時**複製到 **CF** 與 **最高位元（MSB）**。
- **不會遺失任何位元**。

### 圖解

```
   ┌─────────────────────────┐
   ▼                         │
   [ • • • • • • • • ] ──► CF │
   └─────────────────────────┘
   (最低位 → CF，同時繞回最高位)
```

### 例題
```asm
mov al, 11110000b
ror al, 1          ; AL = 01111000b
                   ; 最低位 0 繞回到最高位

mov dl, 3Fh
ror dl, 4          ; DL = F3h
                   ; 3Fh=00111111 → 向右旋轉 4 位 → 11110011 = F3h
```

> 💡 有趣的觀察：`3Fh` 不論是 `rol 4` 還是 `ror 4`，結果都是 `F3h`。
> 因為 8 位元的數字旋轉 4 位，左旋 4 = 右旋 4（剛好是一半）。

# 七、移位與旋轉的應用（Applications）

## 應用 1：二進位乘法（Binary Multiplication）

- 當**乘數是 2 的次方**時，SHL 可高效率完成無號乘法。
- 當**乘數不是 2 的次方**時 → 把乘數**拆成「2 的次方之和」**，分別左移後相加。

### 例題：EAX × 36

把 36 拆解：
```
EAX × 36
= EAX × (32 + 4)
= (EAX × 32) + (EAX × 4)
= (EAX × 2⁵) + (EAX × 2²)
```

```asm
mov eax, 123
mov ebx, eax
shl eax, 5         ; 乘以 2⁵ = ×32
shl ebx, 2         ; 乘以 2² = ×4
add eax, ebx       ; 相加 → EAX = 123 × 36 = 4428
```

> 🔑 **拆解技巧**：把乘數寫成二進位，看哪幾個位元是 1。
> 36 = 100100b → 第 5 位 (2⁵=32) 與第 2 位 (2²=4) 為 1 → 32 + 4 = 36。

---

## 應用 2：簡易加密（Simple Encryption）

- **位元旋轉（bit rotation）** 常用於對稱式加密演算法，用來**混淆資料（obfuscate）**、增加**非線性度（non-linearity）**。
- 加密時旋轉，解密時往反方向旋轉相同位數即可還原。

### 例題（使用 ROL）
```asm
; 假設 EAX 存放原始資料
mov eax, 0x12345678   ; 範例明文 (plaintext)

; 將 EAX 向左旋轉 8 位
rol eax, 8            ; EAX 現在向左旋轉了 8 位

; 旋轉後的結果可作為加密步驟的一部分
```

> 💡 解密：對應使用 `ror eax, 8` 即可還原。

# 🔥 速背重點整理

### 1. 六大指令一覽表

| 指令 | 全名 | 方向 | 補位 / 行為 | 主要用途 |
|------|------|------|------------|---------|
| **SHL** | Shift Left | 左 | 低位補 **0**，高位→CF | 無號 ×2ⁿ |
| **SHR** | Shift Right | 右 | 高位補 **0**，低位→CF | **無號** ÷2ⁿ |
| **SAL** | Shift Arithmetic Left | 左 | 低位補 **0**（=SHL） | 有號 ×2ⁿ |
| **SAR** | Shift Arithmetic Right | 右 | 高位補 **符號位元** | **有號** ÷2ⁿ |
| **ROL** | Rotate Left | 左 | 高位→CF 且繞回低位 | 旋轉、加密 |
| **ROR** | Rotate Right | 右 | 低位→CF 且繞回高位 | 旋轉、加密 |

### 2. 乘除法口訣

```
左移 n 位 = × 2ⁿ        （SHL / SAL）
右移 n 位 = ÷ 2ⁿ        （SHR=無號、SAR=有號）
```

### 3. 最容易混淆的點（必考）

- **無號數除法 → SHR**（補 0）
- **有號數除法 → SAR**（補符號位元，保留正負號）
- **SAL = SHL**（完全相同，左移時無號/有號一致）
- **Shift（移位）會丟位元**到 CF；**Rotate（旋轉）不丟位元**，會繞回另一端。

### 4. 補位規則記憶

```
邏輯 (Logical, SHL/SHR)  → 一律補 0
算術 (Arithmetic, SAR)   → 右移補「符號位元」(保留正負)
旋轉 (Rotate, ROL/ROR)   → 移出的位元繞回去 + 進 CF
```

### 5. 乘數非 2 次方 → 拆成「2 次方之和」

```
EAX × 36 = (EAX × 32) + (EAX × 4)
         = (EAX << 5) + (EAX << 2)
```

### 6. 常見移位結果速查（以 8 位元為例）

| 起始 | 指令 | 結果 | 說明 |
|------|------|------|------|
| 6Bh (0110 1011) | shr 1 | 35h | 邏輯右移補 0 |
| 35h (0011 0101) | shl 3 | A8h | 邏輯左移補 0 |
| 8Ch (1000 1100) | sar 1 | C6h | 算術右移補符號 1 |
| C6h (1100 0110) | sar 3 | F8h | 算術右移補符號 1 |
| 11110000b | rol 1 | 11100001b | 最高位繞回最低位 |
| 11110000b | ror 1 | 01111000b | 最低位繞回最高位 |

---

✅ **本章一句話總結**：移位（Shift）做快速乘除法——左乘右除；無號用 SHR、有號用 SAR；旋轉（Rotate）不丟位元，可用於加密。
