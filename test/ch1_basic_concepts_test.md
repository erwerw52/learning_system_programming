# CH1 基本概念 — 練習題

> 主題：資料表示法 · 布林運算 · x86 架構 ｜ 共 21 題（原題 3・補充 18）

---

### Q01 `原題`

變數位址為 `00400020`，下一個變數位址為 `0040006A`，此變數佔幾個 bytes？

**答案：** `0040006A − 00400020 = 4A₁₆ = 74 bytes`

---

### Q02 `原題`

什麼是 portable（可移植性）？組合語言可移植嗎？

**答案：** portable 指軟體在不同平台／處理器上的支援程度。組合語言**不可移植**，因為它綁定特定的處理器家族。

---

### Q03 `原題`

用傳統編譯工具鏈建置一支 C 程式，需要 compiler、assembler、linker 嗎？請說明。

**答案：** 需要。先用 **compiler** 把 C 編譯成組合語言，再用 **assembler** 轉成機器碼，最後用 **linker** 把多個 object file 連結成一個可執行檔。

### Q04 `補充`

將二進位 `00001001₂` 轉成十進位。

**答案：** (1×2³)+(1×2⁰) = 8 + 1 = **9**

### Q05 `補充`

將十進位 `37` 轉成二進位（重複除以 2）。

**答案：** 37 = 32+4+1 → **100101₂**（餘數由下往上讀）

### Q06 `補充`

將 `3BA4₁₆` 轉成十進位。

**答案：** (3×16³)+(11×16²)+(10×16¹)+(4×16⁰) = 12288+2816+160+4 = **15,268**

### Q07 `補充`

將十進位 `422` 轉成十六進位。

**答案：** 422/16=26 餘 6；26/16=1 餘 A；1/16=0 餘 1 → **1A6₁₆**

### Q08 `補充`

將 `000101101010011110010100₂` 轉成十六進位。

**答案：** 每 4 bits 一組：0001 0110 1010 0111 1001 0100 → **16A794₁₆**

### Q09 `補充`

計算十六進位加法 `6A + 4B`。

**答案：** A+B=21 → 餘 5 進 1；6+4+1=B → **B5₁₆**

---

### Q10 `補充`

計算十六進位減法 `75 − 47`。

**答案：** 5 不夠減 7，借位：16+5=21；21−7=14=E；7−1−4=2 → **2E₁₆**

---

### Q11 `補充`

求 `+1` 的 8-bit 二補數表示（即 −1）。

**答案：** +1 = 00000001 → 反轉 11111110 → +1 → **11111111**（= −1）

---

### Q12 `補充`

計算二進位減法 `00001100 − 00000011`。

**答案：** 12 − 3 = 9 = **00001001**（做法：00001100 + 11111101）

---

### Q13 `補充`

20 bits 能儲存的最大 unsigned 與最大 signed 正數各是多少？

**答案：** Unsigned 最大 = 2²⁰ − 1 = **1,048,575**；Signed 最大正數 = 2¹⁹ − 1 = **524,287**

---

### Q14 `補充`

hex 串 `54 68 65 20 45 6E 64` 用 ASCII 解讀是什麼？

**答案：** T h e (空格) E n d → **"The End"**

---

### Q15 `補充`

下列哪些 hex 數值是負數：`8A, C5, A2, 9D, 6F`？

**答案：** 最高 hex digit > 7 即為負 → **8A、C5、A2、9D**；6F 為正。

---

### Q16 `補充`

寫出 EFLAGS 六個 status flag 各自的觸發條件（CF / OF / SF / ZF / AF / PF）。

**答案：** **CF** unsigned 超出範圍；**OF** signed 超出範圍；**SF** 結果為負；**ZF** 結果為 0；**AF** bit3→bit4 進位；**PF** 結果低位元組中 1 的個數為偶數。

---

### Q17 `補充`

32-bit 暫存器轉 64-bit 的命名規則為何？舉 EAX、EIP 為例。

**答案：** `E` 開頭改成 `R` 開頭：EAX→**RAX**、EIP→**RIP**；另新增 **R8 ~ R15**，共 16 個 64-bit 通用暫存器。

---

### Q18 `補充`

填出多工器 `(Y∧S)∨(X∧¬S)` 在 S=0 與 S=1 時的輸出。

**答案：** S=0 時輸出 = **X**；S=1 時輸出 = **Y**（選擇訊號 S 決定輸出哪一路）。

---

### Q19 `補充`

`¬X ∨ Y` 的運算順序為何？布林運算子的優先序口訣？

**答案：** 先 **NOT（¬X）**，再 **OR**。優先序：**NOT > AND > OR**（括號最大）。

---

### Q20 `補充`

AL（組合語言）與機器語言、與 C++/Java 的對應關係分別是？

**答案：** AL ↔ 機器語言為 **one-to-one**（一對一）；C++/Java ↔ AL 為 **one-to-many**（一對多）。

---

### Q21 `補充`

將 memory hierarchy 由「快到慢」排序，並說明趨勢。

**答案：** Registers > L1 Cache > L2 Cache > Memory(RAM)。**越靠近 CPU 越快、越小**。
